haskell-modbus
==============

An implementation of the Modbus protocol in Haskell.

### Current Features:

Transports:
- Modbus RTU over Serial
- Modbus RTU over TCP

Commands:
- Read Discrete Inputs
- Read Coils
- Write Single Coil
- Write Multiple Coils
- Read Input Register
- Read Holding Registers
- Write Single Register
- Write Multiple Registers
